# Roblox Studio Notes

Anotações pessoais sobre desenvolvimento de jogos no Roblox Studio.

O conteúdo é organizado como um vault do [Obsidian](https://obsidian.md), onde cada nota pode referenciar outras através de links internos — permitindo navegar pelo conhecimento de forma conectada.

## Estrutura

- **Notas temáticas** — resumos sobre conceitos, padrões e técnicas do Roblox
- **Glossário** — definições de termos e APIs usados no desenvolvimento

## Como usar

Clone o repositório e abra a pasta como um vault no Obsidian:

```bash
git clone https://github.com/EaeDave/roblox-studio-notes.git
```

> As anotações são públicas e podem ser úteis para outros desenvolvedores aprendendo Roblox.
